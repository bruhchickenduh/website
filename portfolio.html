Snowman Builder 1.1.9 Project


This game allows the user to build a snowman to their own liking by answering a set of questions. The user can customize the color, size, and accessories the snowman can have. We used both general turtle commands and conditional statements(if and while loops) from the PLTW lessons we went over to create the game.


Whack-A-Turtle 1.2.5 Project


This game is based on Whack-A-Mole but with turtles. The user clicks colored labeled buttons that correspond to a colored turtle to win but their score only increases if they click the buttons when the turtles’ color are active and not black. Black colored turtles means it’s deactivated and will not count the user’s clicks. We used many list indexes and conditional statements to sort out the turtle’s colors and buttons to correspond to when the user clicks correctly. We also manipulated strings when inputting the leaderboard with the user’s name and used functions to reduce duplications.

Understanding Complex Systems-NetLogo Simulation 4.1.4 Project



This project was analyzing a NetLogo Simulation and my group chose a firework one. We observed and analyzed how different fireworks of color and size are shot up and explode with randomly falling fragments. This simulation allowed us to adjust gravity, the maximum number of fireworks, fragments, initial x and y velocities, and fade amounts; but it lacked different backgrounds and only showcased fireworks with no other physical elements to influence the behavior of the fireworks.
